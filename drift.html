Using Google Glass combined with image analysis and machine learning classifiers on geospatial data to implement a 4G tactical Intel system to: Find different types of terrain by identifying objects and/or targets within the space using voice and/or touch controls (depending on the mechanics of the UI) With trained AI algorithms, determine optimal paths to goals, avoiding obstacles, adversaries, and dangerous paths. The final objective will be to fuse these analytics with an Advanced User Experience device (Google Glass, Oculus Rift) to display instructions for agents in real time. This will be split into 3 efforts run in parallel: Machine Learning Classification of geospatial map data Artificial Intelligence path fining implementation Development of head-up display for Advanced User Interface devices that takes instructions from the AI system (leveraging existing APIs from the device) subject to voice control and directions from the wearer
